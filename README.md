# 1. Клонируйте репозиторий:cmd
```bash
    git clone -b master https://github.com/mcsaltine02/Website-for-the-children-s-railway

    cd Website-for-the-children-s-railway
```

# 2. Получите свой токен VK

### Шаг 1: Создать приложение (Сайт / Мини-приложение)

1. Перейдите:  
   https://vk.com/editapp?act=create
2. Выберите **Сайт** или **Мини-приложение**.
3. Заполните:
   - Название: (любое). Пример: `VkPostApp` 
   - Платформы: `Web-приложение`
   - Базовый домен: `localhost`
   - Доверенный Redirect URL: `https://oauth.vk.com/blank.html`

### Шаг 2: Получить токен

1. Ниже будет `Ключи доступа`, скопируйте `Сервисный ключ доступа` он же Токен

### Шаг 3: Проверить токен (необязательно, но желательно)
`Измените` url подставив вместо `ВАШ_ТОКЕН` на свой токен.
В этвете должен быть json - всё работает
```bash
curl "https://api.vk.com/method/wall.get?owner_id=-2608975&count=10&access_token=ВАШ_ТОКЕН&v=5.199"
```


# 3. Установите переменную окружения:cmd
(Windows)
```bash для Windows
    setx VK_ACCESS_TOKEN=ваш_токен
```
(Linux)
```bash для Linux
    export VK_ACCESS_TOKEN='ваш_токен'
``` 

### 3.2 Проверьте, что переменная установлена:cmd
```bash
    echo %VK_ACCESS_TOKEN%
``` 

# 4. Установка необходимого для запуска
#### 4.1. Установите Java 17 или выше
- Перейдите по ссылке: [Java Downloads](https://www.oracle.com/java/technologies/downloads/)
- Установите Java, следуя инструкциям на сайте.
- Настройте переменную окружения `JAVA_HOME`:
  - Для этого найдите папку, где установлена Java, и укажите ее путь в переменной `JAVA_HOME`.

- **Проверьте корректность установки**:
  Откройте командную строку (cmd) и выполните:
  ```bash
  java -version
  ```
  Вы должны увидеть информацию о версии Java.

#### 4.2. Скачайте дистрибутив Maven
- Откройте сайт: [Apache Maven Downloads](https://maven.apache.org/download.cgi) и скачайте Maven (обычно это zip-архив).
- Распакуйте архив Maven в удобную папку на вашем компьютере.

#### 4.3. Настройте переменные среды Windows
- Создайте или обновите переменные `M2_HOME` и `MAVEN_HOME`, указав путь к папке с Maven (например, `C:\path\to\apache-maven-x.x.x`).
- Добавьте в системную переменную `PATH` путь к папке `bin` внутри установленного Maven, например:
  ```plaintext
  %M2_HOME%\bin
  ```
- Убедитесь, что в `PATH` также добавлен путь к Java JDK (например, `%JAVA_HOME%\bin`).

#### 4.4. Проверьте установку
- Откройте командную строку (cmd) и выполните:
  ```bash
  mvn -version
  ```
  Эта команда должна вывести версию Maven, версию Java и информацию об операционной системе.

# 5. Запуск проекта
После успешной проверки установки вы можете запускать проект, выполнив следующие команды в командной строке:
```bash
mvn clean install
mvn spring-boot:run
```
